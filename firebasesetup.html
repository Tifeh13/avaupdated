Firebase Setup Guide

Ava Admin ¬∑ Real-Time Order Collection ¬∑ Step-by-Step
What this does: Every customer who checks out on your site (from any device, anywhere in the world) will instantly appear in your admin panel. Firebase is free for your usage level and takes about 10 minutes to set up.
Step 1

Create a Firebase Project

  • Go to console.firebase.google.com
  • Click "Create a project"
  • Name it ava-jewelry (or anything you like)
  • Disable Google Analytics (not needed) ‚Üí Click "Create project"
  • Wait ~30 seconds, then click "Continue"
Step 2

Create a Firestore Database

  • In the left sidebar, click "Firestore Database" (under Build)
  • Click "Create database"
  • Select "Start in test mode" ‚Üí Click Next
  • Choose your closest region (e.g. us-central or europe-west) ‚Üí Click "Enable"
‚ö†Ô∏è Test mode expires in 30 days. Before it expires, go to Firestore ‚Üí Rules and change allow read, write: if false to allow read, write: if true to keep it open. (Fine for a private store.)
Step 3

Register Your Web App & Get Config Keys

  • In Firebase console, click the gear icon ‚öôÔ∏è ‚Üí "Project settings"
  • Scroll to "Your apps" ‚Üí Click the </> Web icon
  • Name it ava-web ‚Üí Click "Register app"
  • You'll see a config block like this ‚Äî copy it:
const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "ava-jewelry.firebaseapp.com",
  projectId: "ava-jewelry",
  storageBucket: "ava-jewelry.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123..."
};
‚úÖ Keep this config block. You'll paste it into both payment.html and admin.html in the next step.
Step 4

Update Your Files

Open each file I gave you and find the line that says:

// ‚ñº‚ñº‚ñº PASTE YOUR FIREBASE CONFIG HERE ‚ñº‚ñº‚ñº

Replace those lines with your actual config from Step 3. Do this in both:

  • payment.html ‚Äî so orders get saved when customers pay
  • admin.html ‚Äî so you can see orders in the dashboard
Step 5

Push to GitHub & Deploy on Vercel

  • Save all files
  • In VS Code terminal, run:
git add .
git commit -m "Add Firebase order tracking"
git push
  • Vercel will auto-deploy in ~1 minute
  • Open your live site and do a test checkout
  • Then open your admin panel ‚Äî the order should appear immediately! üéâ
